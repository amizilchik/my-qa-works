# Чек-лист: Функциональность слайдера на странице COMPANIES

## Предусловия:
- [ ] Пользователь авторизован
- [ ] Открыта страница "COMPANIES"
- [ ] Слайдер отображается на странице

## Навигация по номерам страниц:
- [ ] Отображаются номера страниц
- [ ] Клик по номеру страницы переключает слайдер
- [ ] Активная страница выделена
- [ ] Переключение работает для всех страниц
- [ ] Переключение работает стабильно (не через раз)

## Навигация стрелками:
- [ ] Левая стрелка отображается
- [ ] Правая стрелка отображается
- [ ] Левая стрелка переключает на предыдущую страницу
- [ ] Правая стрелка переключает на следующую страницу
- [ ] Стрелки неактивны на первой/последней странице (если должно быть так)
- [ ] Стрелки работают стабильно при многократном клике

## Отображение контента:
- [ ] Контент каждой страницы отображается корректно
- [ ] Изображения загружаются
- [ ] Текст отображается без обрезания
- [ ] Ссылки работают
- [ ] Нет наложений контента

## Ошибки:
- [ ] Нет критических ошибок в консоли браузера
- [ ] Страница не зависает при переключении
- [ ] Нет дублирования страниц
- [ ] Нет пропущенных страниц
- [ ] Нет визуальных артефактов
