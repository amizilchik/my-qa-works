# Чек-лист: Добавление заметки к контакту

## Предусловия:
- [ ] Пользователь авторизован
- [ ] Открыт профиль контакта

## Функциональность:
- [ ] Поле "Add a note" отображается
- [ ] Поле принимает текстовый ввод
- [ ] Кнопка "ADD THIS NOTE" присутствует
- [ ] Кнопка активна при вводе текста
- [ ] Заметка отображается сразу после добавления
- [ ] Заметка сохраняется после перезагрузки страницы
- [ ] Можно добавить несколько заметок
- [ ] Заметки отображаются в хронологическом порядке

## Валидация:
- [ ] Пустая заметка не добавляется
- [ ] Заметка из пробелов не добавляется
- [ ] Длинная заметка обрабатывается корректно
- [ ] Спецсимволы обрабатываются корректно

## Ошибки:
- [ ] Нет дублирования заметок
- [ ] Нет исчезновения заметок после добавления
- [ ] Нет ошибок в консоли браузера
