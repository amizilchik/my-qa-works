# Чек-лист: Форма добавления контакта

## Предусловия:
- [ ] Пользователь авторизован
- [ ] Открыта страница "CONTACTS"
- [ ] Нажата кнопка "New contact"

## Обязательные поля:
- [ ] First Name - поле присутствует
- [ ] Last Name - поле присутствует
- [ ] Email - поле присутствует (если есть)
- [ ] Phone - поле присутствует (если есть)

## Валидация First Name:
- [ ] Принимает буквы (латиница/кириллица)
- [ ] Не принимает спецсимволы
- [ ] Не принимает числа (если не должно)
- [ ] Не принимает пустое значение
- [ ] Не принимает только пробелы
- [ ] Ограничен по длине (если есть ограничение)

## Валидация Last Name:
- [ ] Принимает буквы (латиница/кириллица)
- [ ] Не принимает спецсимволы
- [ ] Не принимает числа (если не должно)
- [ ] Не принимает пустое значение
- [ ] Не принимает только пробелы
- [ ] Ограничен по длине (если есть ограничение)

## Кнопка Save:
- [ ] Активна при заполнении обязательных полей
- [ ] Неактивна при незаполненных полях
- [ ] Создает контакт при корректных данных
- [ ] Показывает ошибки при некорректных данных

## Ошибки:
- [ ] Нет критических ошибок при сохранении
- [ ] Нет дублирования контактов
- [ ] Нет ошибок в консоли браузера
