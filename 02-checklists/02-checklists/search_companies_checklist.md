# Чек-лист: Поиск компаний

## Предусловия:
- [ ] Пользователь авторизован
- [ ] Открыта страница "COMPANIES"

## Функциональность:
- [ ] Поле поиска отображается
- [ ] Поиск работает при вводе текста
- [ ] Результаты отображаются динамически или после Enter
- [ ] При пустом поиске отображаются все компании
- [ ] При отсутствии результатов показывается сообщение
- [ ] Поиск работает с кириллицей
- [ ] Поиск работает с латиницей
- [ ] Поиск не чувствителен к регистру (если должен быть)

## Валидация:
- [ ] Поиск с 1 символом работает
- [ ] Поиск с 2+ символами работает
- [ ] Поиск со спецсимволами обрабатывается корректно
- [ ] Поиск с пробелами обрабатывается корректно
- [ ] Поиск с числами работает

## Ошибки:
- [ ] Нет критических ошибок при поиске
- [ ] Страница не зависает при поиске
- [ ] Нет ошибок в консоли браузера
- [ ] Нет дублирования результатов
