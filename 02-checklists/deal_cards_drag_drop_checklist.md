# Чек-лист: Drag-and-Drop плашек сделок

## Предусловия:
- [ ] Пользователь авторизован
- [ ] Открыта страница "DEALS"

## Базовая функциональность:
- [ ] Плашки можно перетаскивать между колонками
- [ ] Плашки можно перетаскивать внутри колонки
- [ ] При перетаскивании показывается визуальная обратная связь
- [ ] Плашка возвращается на место при отмене перетаскивания

## Скорость перетаскивания:
- [ ] Медленное перетаскивание работает корректно
- [ ] Быстрое перетаскивание работает корректно
- [ ] Хаотичное перетаскивание работает корректно
- [ ] Перетаскивание нескольких плашек подряд работает

## Визуальное отображение:
- [ ] Плашка выделяется при начале перетаскивания
- [ ] Показывается место назначения при перетаскивании
- [ ] Анимация перетаскивания плавная
- [ ] Нет визуальных артефактов

## Сохранение позиций:
- [ ] Позиции сохраняются после перезагрузки страницы
- [ ] Позиции сохраняются после перехода на другую страницу и возврата
- [ ] Нет дублирования плашек
- [ ] Нет пропажи плашек

## Ошибки:
- [ ] Нет ошибок в консоли браузера
- [ ] Страница не зависает при перетаскивании
- [ ] Нет конфликтов при одновременном перетаскивании
