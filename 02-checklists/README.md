## Описание
Практические чек-листы для тестирования реальной CRM-системы [dev-crm.qa-playground.com](https://dev-crm.qa-playground.com/#/). Включает smoke-тестирование и функциональные проверки.

## Содержание

###  Smoke-тестирование
- **smoke-test-checklist.xlsx** - Базовая проверка работоспособности приложения
- **Покрытие:** доступность системы, аутентификация, основные модули, навигация
- **Цель:** быстрая проверка после деплоя или перед глубоким тестированием

###  Функциональное тестирование
- **test-cases-check-list.xlsx** - Детальные проверки функциональности CRM
- **Покрытие:** основные сценарии использования, позитивные и негативные тесты
- **Модули:** контакты, сделки, задачи, настройки системы

###  Дополнительные тест-кейсы
- **тест-кейсы-чек-лист.xlsx** - Расширенная коллекция тестовых сценариев
- **Особенности:** граничные значения, эквивалентное разделение, бизнес-логика
