## Описание
Коллекция чек-листов для smoke-тестирования и проверки основных функциональностей.

##  Цель
Демонстрация навыков:
- Создания эффективных чек-листов
- Приоритизации критических функциональностей
- Организации процесса тестирования
- Быстрой проверки основных сценариев

## Особенности
- Четкая структура проверок
- Указание окружения и версий
- Отметки о статусе выполнения
- Возможность быстрого повторного использования
