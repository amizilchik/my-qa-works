### Summary
Слайдер на странице "COMPANIES" работает некорректно: переключение по номерам — через раз, стрелки — не работают

### Pre-conditions
- Пользователь авторизован
- Открыта страница "COMPANIES"

### Environment
- Браузер: Chrome 118  
- ОС: Windows 11  
- URL: https://dev-crm.qa-playground.com/

### Steps to reproduce
1. Перейти в раздел “COMPANIES“  
2. Прокрутить страницу вниз до слайдера  
3. Попробовать переключать страницы по номерам  
4. Попробовать переключать страницы через боковые стрелки

### Expected result
- Слайдер корректно переключает страницы как по номерам, так и через боковые стрелки

### Actual result
- Переключение по номерам работает через раз  
- Боковые стрелки не реагируют на клики

### Reproducibility
Always

### Severity
Medium

### Priority
Medium
