# Тест-кейсы: Валидация полей First Name и Last Name при добавлении контакта

| ID  | Название теста                         | Предусловие              | Шаги                                           | Ожидаемый результат                        |
|-----|----------------------------------------|---------------------------|------------------------------------------------|--------------------------------------------|
| TC01| Ввод валидного имени и фамилии         | Авторизован, на странице CONTACTS | 1. Нажать "New contact" 2. Ввести валидные имя и фамилию 3. Нажать "Save" | Контакт успешно создаётся |
| TC02| Ввод пустых значений                   | Авторизован, на странице CONTACTS | 1. Нажать "New contact" 2. Оставить поля First Name и Last Name пустыми 3. Нажать "Save" | Поля подсвечиваются, ошибка валидации |
| TC03| Ввод спецсимволов                       | Авторизован, на странице CONTACTS | 1. Нажать "New contact" 2. Ввести спецсимволы в First Name и Last Name 3. Нажать "Save" | Поля подсвечиваются, ошибка валидации |
| TC04| Ввод чисел                             | Авторизован, на странице CONTACTS | 1. Нажать "New contact" 2. Ввести числа в First Name и Last Name 3. Нажать "Save" | Поля подсвечиваются, ошибка валидации |
| TC05| Ввод только пробелов                   | Авторизован, на странице CONTACTS | 1. Нажать "New contact" 2. Ввести пробелы в First Name и Last Name 3. Нажать "Save" | Поля подсвечиваются, ошибка валидации |
| TC06| Ввод кириллицы                         | Авторизован, на странице CONTACTS | 1. Нажать "New contact" 2. Ввести имя и фамилию на кириллице 3. Нажать "Save" | Контакт успешно создаётся |
| TC07| Ввод очень длинного имени              | Авторизован, на странице CONTACTS | 1. Нажать "New contact" 2. Ввести >100 символов 3. Нажать "Save" | Ошибка валидации или обрезка данных |
