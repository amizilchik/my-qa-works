# Тест-кейсы: Поиск компаний

| ID  | Название теста                         | Предусловие              | Шаги                                           | Ожидаемый результат                        |
|-----|----------------------------------------|---------------------------|------------------------------------------------|--------------------------------------------|
| TC01| Поиск по валидному названию компании   | Авторизован, на странице COMPANIES | 1. Ввести корректное название компании 2. Нажать Enter или подождать | Отображаются подходящие результаты |
| TC02| Поиск по части названия                | Авторизован, на странице COMPANIES | 1. Ввести часть названия 2. Нажать Enter | Отображаются компании, содержащие этот текст |
| TC03| Поиск по несуществующему названию      | Авторизован, на странице COMPANIES | 1. Ввести несуществующее название 2. Нажать Enter | Отображается сообщение "Ничего не найдено" |
| TC04| Поиск с пустым полем                   | Авторизован, на странице COMPANIES | 1. Оставить поле поиска пустым 2. Нажать Enter | Отображается полный список компаний |
| TC05| Поиск с вводом спецсимволов            | Авторизован, на странице COMPANIES | 1. Ввести спецсимволы (например, “$$”) 2. Нажать Enter | Отображаются подходящие результаты или сообщение об ошибке |
| TC06| Поиск с вводом 1 символа               | Авторизован, на странице COMPANIES | 1. Ввести 1 символ 2. Нажать Enter | Отображаются компании, содержащие этот символ |
| TC07| Поиск с вводом более 2 символов        | Авторизован, на странице COMPANIES | 1. Ввести 2+ символа 2. Нажать Enter | Отображаются подходящие результаты |
| TC08 | Поиск с использованием кириллицы | Авторизован, на странице COMPANIES | 1. Ввести "Компания" 2. Нажать Enter | Отображаются подходящие результаты |
