#  Тест-кейсы

## Описание
В этой папке собраны примеры тест-кейсов для различных функциональностей веб-приложений.

##  Цель
Демонстрация навыков:
- Разработки комплексных тест-кейсов
- Применения техник тест-дизайна
- Формирования четких шагов воспроизведения
- Определения ожидаемых результатов

## Используемые техники
- Анализ граничных значений
- Эквивалентное разделение
- Предусловия/постусловия
- Приоритизация тестов
